LotWizard (Capstone)

Web

Capstone project. Team-based, following the Agile methodology. Written in JavaScript using React, Next.js, Node, Tailwind CSS, serverless mySQL, and connected to eBay API.

LotWizard is a platform where trucking companies can sponsor and reward drivers for good performance. It uses an eBay API to support a sponsor's catalog in which drivers can use in-app currency, or points, to make purchases. LotWizard supports 3 different types of users - drivers, sponsors, and admins - each with different levels of access/control.
